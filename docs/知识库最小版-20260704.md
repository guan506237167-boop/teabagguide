# 养生茶包 / Herbal Tea Bag Guide 知识库最小版

更新时间：2026-07-04

## 站点类型

产品站

## MVP 目标

先验证主题、工具或产品展示方向、栏目结构、视觉风格和后续商业承接位置。不在本阶段追求大量文章或真实联盟商品填满。

## 必备栏目

- Home
- Tools / Products
- Guide
- Categories
- FAQ
- About
- Contact
- Article detail

## 内容标准

- 正式发布文章不低于 1000 英文词。
- 开头直接回答用户问题，不写空泛铺垫。
- 每个 H2 都要有具体例子或判断标准。
- 保留 SEO/GEO 结构，但优先保证人读起来顺。
- 禁止只写三四百词的短薄页面。

## 图片标准

- 首页必须有首屏视觉图。
- 核心栏目必须有视觉锚点。
- 文章详情页必须有封面图。
- 图片固定比例，不能拉伸变形。
- 文字不能压在低对比度图片上。

## 商业化预留

- 广告联盟
- 联盟产品
- 虚拟报告 / PDF / 模板
- 邮件订阅
- 后续服务或课程入口

## 上线前补齐

- 域名和 canonical
- GA
- GSC 验证文件
- Ahrefs
- 301 非 www 到 www
- robots.txt / sitemap.xml / llms.txt 复查
- Cloudflare SSL 和 WAF 基础规则
- 首批正式文章
